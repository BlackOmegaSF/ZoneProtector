
# Zone Protector 
This mod protects player-defined zones from hostile mobs.

## Available commands:

- `/zone list`
- `/zone protect <x1> <y1> <z1> <x2> <y2> <z2> [<name>]`
- `/zone unprotect by_name <name>`
- `/zone unprotect by_coords <x1> <y1> <z1> <x2> <y2> <z2>` 



## Protections:

Inside a protected zone:
- Players cannot use spawn eggs
- Endermen and shulkers cannot teleport in
- Hostile mobs cannot spawn
- TNT cannot destroy blocks
